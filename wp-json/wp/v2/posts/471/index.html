Python is a versatile and powerful programming language that can be used for various tasks, such as web development, data analysis, machine learning, and more. One of the common operations that we may need to perform with strings is to check if a string contains a certain word or substring.<\/p>\n\n\n\n

Check if a string contains substring in python<\/h2>\n\n\n\n

The most intuitive ways to check if a string contains a substring in Python is to use the in<\/code> operator. It returns True<\/code> if the substring is found in the string, and False<\/code> otherwise. For example: the expression \"mother\" in \"my grandmother\"<\/code> returns True<\/code> when printed.<\/p>\n\n\n\n

substring = "python"\nstring = "I love python programming"\nprint(substring in string) #True\nprint("program" in string) #True<\/pre><\/div>\n\n\n\n
To check if a string contains a substring case-insensitive, apply the lower()<\/code> methods on both strings and then use the in<\/code> operator. For instance : substring.lower() in string.lower()<\/code> or \"dad\".lower() in \"grandDad\".lower()<\/code>. This may or may not be what we want, depending on our use case. If we only want to match whole words, we may need the next method.<\/p>\n\n\n\n
Check if a string contains a word in python<\/h2>\n\n\n\n
To check if a string contains a single word in Python, use the in<\/code> operator to see if that word is in the list<\/strong> returned by the split()<\/code> method applied directly on your string. For instance:   \"father\" in \"my grandfather\".split()<\/code> results in False<\/code> because the returned list of words is ['my', 'grandfather']<\/code>.<\/p>\n\n\n\n
word = "program"\nstring = "We love python programming"\nwordList = string.split()\n\nprint(wordList) #['We', 'love', 'python', 'programming']\n\n# Check if a string contains a word\nprint(word in wordList) #False<\/pre><\/div>\n\n\n\n
This method<\/a> will only match whole words, since it splits the string by whitespace characters. However, if you want to split words in your string by a comma (or a delimiter), just passed that only one<\/strong> to the split()<\/code> method, such as split(',')<\/code>. If you want to pass more than one delimiter, use the re.split()<\/code> method with a regex instead.<\/p>\n\n\n\n
Check if a string contains a regex python<\/h2>\n\n\n\n
To use regular expressions in Python, we need to import re<\/code> module. Then called re.search(pattern,string)<\/code>, the first argument is a regex or a substring, and the second one is the full string. It returns a Match object<\/strong> if the pattern is found in the string, or None<\/code> otherwise. For example: re.search(\"mom\",\"dad\")<\/code> returns None<\/code>.<\/p>\n\n\n\n
To check if a string contains a word using regular expressions, we can use re.search() with an if statement, and convert the result to a boolean value using bool()<\/code>. For example:<\/p>\n\n\n\n
import re\n\nbool(re.search("world", "hello world")) # True\nbool(re.search("java", "hello world")) # False<\/pre><\/div>\n\n\n\n
One of the advantages of using regular expressions is that we can specify more details<\/a> about how we want to match the word. For example, we can use special characters and modifiers to indicate case sensitivity, boundaries, repetitions, alternatives, etc.<\/p>\n\n\n\n
If we want to match only whole words<\/strong> that are case insensitive, we can use the \\b<\/code> metacharacter to mark word boundaries, and the re.IGNORECASE<\/code> flag to ignore case differences. For example: re.search(r\"\\bpython\\b\", \"Python\", re.IGNORECASE)<\/code> returns Match while re.search(r\"\\bpy\\b\", \"Python\", re.IGNORECASE)<\/code> returns None.<\/p>\n\n\n\n
import re\nstring = "We love python"\n\nprint (re.search(r"\\bpython\\b", string, re.IGNORECASE))\n# Output: <re.Match object; span=(8, 14), match='python'>\n\nprint(re.search(r"\\bPython\\b", string))\n# Output: None<\/pre><\/div>\n\n\n\n
You may also note that there is a r<\/code> character before the pattern in the above example. It stands for raw string in Python, which treats the backslash character (\\) as a literal character. If this knowledge is new to you, then you should learn more about regular expressions also. They are a powerful tool for pattern matching and text processing, that can handle complex and varied scenarios.<\/p>\n\n\n\n
